# Landing page de Shopp para GitHub Pages

## Archivos incluidos

- `docs/index.html`: página principal.
- `docs/privacy.html`: política de privacidad.
- `docs/support.html`: página de soporte.

## Antes de publicar

Busca `tu-email@dominio.com` en los archivos y sustitúyelo por tu correo real.

## Publicar con GitHub Pages

1. Copia la carpeta `docs` a la raíz de tu repositorio.
2. Sube los cambios a la rama `main`.
3. En GitHub abre `Settings` → `Pages`.
4. En `Build and deployment`, selecciona `Deploy from a branch`.
5. Elige la rama `main` y la carpeta `/docs`.
6. Pulsa `Save`.

GitHub mostrará la URL pública cuando termine el despliegue.
