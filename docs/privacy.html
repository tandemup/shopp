Política de privacidad de Shopp

Última actualización: 10 de junio de 2026

Shopp es una aplicación para crear listas de compra, organizar productos, escanear códigos de barras y consultar información relacionada con productos.

1. Datos guardados en el dispositivo

La app puede guardar localmente información introducida por el usuario, como listas de compra, productos, categorías, tiendas favoritas, preferencias e historial de escaneos.

2. Cámara

Shopp puede solicitar acceso a la cámara para leer códigos de barras. La cámara se utiliza para realizar el escaneo solicitado por el usuario.

3. Ubicación

Si utilizas funciones de proximidad de tiendas, la app puede solicitar permiso para acceder a la ubicación del dispositivo. Revisa este apartado y elimínalo si la versión publicada no utiliza ubicación.

4. Servicios externos

Cuando el usuario decide buscar información sobre un producto, Shopp puede abrir servicios externos o buscadores. El uso de esos servicios se rige por sus propias políticas de privacidad.

5. Eliminación de datos

El usuario puede eliminar los datos locales desde las opciones disponibles en la app, limpiando el almacenamiento de la aplicación desde los ajustes de Android o desinstalando la app.

6. Contacto

Para consultas sobre privacidad, escribe a tu-email@dominio.com.
